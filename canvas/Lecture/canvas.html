Demo1 => 线 & 三角形 & 样式
  • 1. moveTo JavaScript 语法: context.lineTo(x,y);
  • 2. lineTo JavaScript 语法: context.moveTo(x,y);
  • 3. stroke JavaScript 语法: context.stroke();
  • 4. fill JavaScript 语法: context.fill();
  • 5. fillStyle JavaScript 语法: context.fillStyle=color|gradient|pattern;
  • 6. strokeStyle JavaScript 语法: context.strokeStyle=color|gradient|pattern;
  • 7. createLinearGradient JavaScript 语法: context.createLinearGradient(x0,y0,x1,y1);
  • 8. createRadialGradient JavaScript 语法: context.createRadialGradient(x0,y0,r0,x1,y1,r1);
  • 9. createPattern JavaScript 语法: context.createPattern(image,"repeat|repeat-x|repeat-y|no-repeat");
  • 10. shadowColor JavaScript 语法: context.shadowColor=color;
  • 11. shadowBlur JavaScript 语法: context.shadowBlur=number;
  • 12. shadowOffset JavaScript 语法: context.shadowOffsetX=number; & context.shadowOffsetY=number;
浏览器不支持canvas，请升级浏览器。
